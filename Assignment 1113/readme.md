* 这次的选题有个比上一次的选题更折磨人，**bug太多了**：国内司法数据公开度本来就差，又涉及未成年人犯罪的档案封存制度；少年法庭审理未成年人案件不公开；大部分时候犯罪的未成年人甚至不会走上法庭。大连少年这样的选题，适合仔仔细细讲故事，也适合定性地追问成因，探讨影响，寻求出路————唯独不太适合从定量上阐述现状。
